# UI Visual Baseline Reference
## Golden Master Layout State

This directory is reserved for the official visual baseline screenshots of the SpeedMath application. These images serve as the "Golden Master" reference to prevent layout drift.

### Required Baseline Screenshots:
1.  **ui_baseline_phone_standard.png**: Phone Portrait — Standard Mode.
2.  **ui_baseline_phone_qmm.png**: Phone Portrait — QMM Mode.
3.  **ui_baseline_phone_modifiers.png**: Phone Portrait — Multiple Modifiers Visible.
4.  **ui_baseline_tablet_portrait.png**: Tablet Portrait — Standard Mode.
5.  **ui_baseline_tablet_landscape.png**: Tablet Landscape — Standard Mode.

### Verification Criteria:
Future UI modifications must be compared against these images to ensure:
*   The **Center Circle** position remains static.
*   The **Modifier Orbit** alignment is preserved.
*   The **Answer Grid** anchor point does not shift.
*   The **Top Status Band** layout remains consistent.
